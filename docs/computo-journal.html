About Computo

Aims and Scope

Computo was created in response to the reproducibility crisis in science, aiming to promote computational/algorithmic contributions in statistics and machine learning (ML) that provide insight into which models or methods are most appropriate for specific scientific questions.

The journal welcomes: - New methods with original stats/ML developments, or numerical studies illustrating theoretical results - Case studies or surveys on stats/ML methods for specific data analysis questions, including neutral comparison studies - Software/tutorial papers presenting implementations of stats/ML algorithms, with concrete use cases and benchmarking

Prospective authors can contact the editor at computo@sfds.asso.fr for pre-submission enquiries.

Open Access and Reproducibility

Computo is free for readers and authors. All content is open access, in line with the Budapest Open Access Initiative. Reproducibility of numerical results is required for publication. Submissions must include all necessary data and code, and the quality of code is assessed during review. Reviews are open and visible after acceptance.

Submit

Submission Overview

Submissions require both scientific content (equations, code, figures, data) and proof of reproducibility. This is achieved by: - Using a notebook system (e.g., Quarto, Rmarkdown, Jupyter) - Providing a virtual environment for dependencies - Setting up continuous integration (CI) for automated builds - Storing data files on platforms like Zenodo or OSF if needed

A submission is a git repository containing: - The notebook source file (Markdown with YAML metadata) - Auxiliary files (BibTeX, figures, data tables) - Environment configuration files - CI configuration files

The compiled notebook (HTML and PDF) is generated via CI and published to a web page. The PDF and repository URL are then submitted via OpenReview.

Templates

Templates are available for R, Python, and Julia. Authors are encouraged to use these to ensure reproducibility.

Reviewing and Publication

Papers are reviewed by external reviewers. Authors should suggest four potential referees. Accepted papers are published under CC BY 4.0.

Code of Ethics for Authors

  • Originality, conflicts of interest, impartiality, funding, and copyright compliance are required.

Review

Review Process

Computo uses Open Review for peer review. Reviews and discussion are made available on the website after acceptance. Reviewers can remain anonymous.

Guidelines for Evaluation

  1. Is the paper within Computo’s scope?
  2. Is the paper clearly written?
  3. Is the paper correct?
  4. Is the paper adequately evaluated?
  5. Is the paper reproducible?
